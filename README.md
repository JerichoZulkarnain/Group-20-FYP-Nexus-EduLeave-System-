# Nexus EduLeave System

Nexus EduLeave System is a web-based leave management portal designed to streamline the process of student leave applications and lecturer approvals. The system features separate dashboards for Students, Lecturers and Administrators, with automated email notifications.

---

## 🚀 Key Features

*   **Student Dashboard:** Apply for leaves, upload supporting documents, track application status, and view leave history.
*   **Lecturer Dashboard:** Review pending leave applications, approve/reject requests with remarks, and maintain logs.
*   **Admin Dashboard:** Manage students and lecturers, track system-wide analytics, and view overall absence summaries.
*   **Email Notifications:** Automated email updates sent via secure SMTP (Gmail) whenever leave applications are updated.

---

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3, Javascript
*   **Backend:** PHP (OOP & Procedural)
*   **Database:** MySQL (Relational)
*   **Email Engine:** SMTP / PHPMailer (Gmail API)
